# Contributing to ProjectionQuery

Thank you for your interest in contributing! We welcome contributions of all kinds: bug fixes, improvements, examples, or documentation.

## How to Contribute

1. **Fork the repository** and create a branch for your feature or bugfix.
2. **Write clear commit messages** and follow existing code style.
3. **Submit a Pull Request (PR)** with a clear description of your changes.

## Guidelines

* Ensure your code builds and passes tests.
* Document new features where applicable.
* Keep contributions focused on a single concern.

## Reporting Issues

Please use the **Issues** tab to report bugs, suggest features, or ask questions.

## Code of Conduct

By participating, you agree to follow the [Code of Conduct](./CODE_OF_CONDUCT.md).
